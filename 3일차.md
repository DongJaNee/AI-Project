### GGUF
- 딥러닝 모델 파일 포맷으로 주로 LLM 추론 앱을 개발할 때 사용.
- 모델의 배포와 사용이 간편하며, On-Device AI와 같이 제한된 자원 환경에서도 좋은 추론 성능을 낼 수 있음.

**장점** 
- 빠른 응답 속도
- HW 호환성
- CPU/GPU 모두 지원
